# InkTime — Smartwatch PCB Design

InkTime este un smartwatch ieftin și open source, proiectat în cadrul cursului TSC (Testarea Sistemelor de Calcul) la Universitatea Politehnica București, 2025-2026.

---

## Descriere generală

Proiectul constă în proiectarea completă a unui PCB pentru un smartwatch cu ecran e-paper, pornind de la specificațiile primite și ducând designul până la stadiul de producție în masă (Engineering Validation — EVT).

Placa are dimensiunile **46mm × 35mm**, grosime **1mm**, și este realizată pe **4 straturi** (Top, GND, Power, Bottom).
